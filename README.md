# Habit Tracker
Live Demo — [Ознакомиться](https://bobicentr.github.io/habit-tracker)

# О проекте
Приложение для отслеживания привычек на React с использованием Feature Sliced Design.  
Позволяет добавлять и удалять привычки, отмечать выполнение по дням. Показывает прогресс за текущую неделю.
Логику по работе с датами и проверкой серий вынес отдельными модулями, чтобы компоненты оставались чистыми и понятными.

# Технологии
- React + Vite — фронтенд  
- JavaScript (ES6+) — логика  
- CSS — стилизация  
- Feature Sliced Design — организация кода  
- LocalStorage — хранение состояния привычек  

# Как запустить
1. Склонировать репозиторий
```
git clone https://github.com/bobicentr/habit-tracker.git
cd habit-tracker
```

2. Установить зависимости
```
npm install
```

3. Запустить проект
```
npm run dev
```

# Что реализовано
- Добавление и удаление привычек  
- Отметка выполнения по дням с проверкой будущих дат  
- Подсчёт серии дней
- Прогресс за текущую неделю  
- Чистая организация кода через Feature Sliced Design  
- Логика работы с датами вынесена в отдельные модули  
- Состояние хранится в LocalStorage
